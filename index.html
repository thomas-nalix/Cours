Les deux type de balises

Il existe 2 types de balises :

  1. Les balises usuelles encadrent du code, par exemple : <p> et </p>.
  2. Les balises dites orphelines n'encadrent aucun code; elles se ferment elles-mêmes. Par exemple : <br/> abrévation de break row ou <hr/> abrévation de horizontal row.
